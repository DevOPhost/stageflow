# Notas de Preview

Checklist usado para revisar a versão pública:

- `index.html` funciona como landing page.
- As telas internas carregam a partir de `/pages`.
- A navegação lateral usa links relativos.
- Os registros iniciais vêm de `demoRecords.js`.
- O tema claro/escuro fica persistido no navegador.
- Relatórios geram prévia visual e PDF.
- A logo da UNIC aparece de forma discreta no contexto acadêmico.
