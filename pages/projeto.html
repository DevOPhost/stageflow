StageFlow

Sistema web para gestão de estágios, acompanhamento de atividades, frequência, carga horária e relatórios acadêmicos em uma experiência estática preparada para GitHub Pages.

Os dados desta versão são simulados para apresentação pública. O projeto principal desenvolvido durante o estágio supervisionado é privado, pois envolve rotinas internas, decisões de negócio e informações institucionais que não devem ser publicadas.

UNIC - Universidade de Cuiabá Estágio Supervisionado Obrigatório Ciência da Computação · Cuiabá/MT
Contexto

Uma versão pública fiel ao fluxo de trabalho, sem expor dados reais.

O StageFlow organiza em uma única interface informações que normalmente ficam distribuídas entre planilhas, documentos, fichas de acompanhamento e relatórios. A versão publicada aqui foi preparada para demonstrar arquitetura, interface e experiência de uso sem depender de backend.

A aplicação usa registros de apresentação com nomes, telefones DDD 65, datas entre 20/08/2025 e 17/09/2025 e atividades coerentes com uma rotina de estágio em Tecnologia da Informação.

Dashboard analítico

Métricas, progresso de horas, frequência, atividades recentes e visão consolidada do estágio.

Cadastros acadêmicos

Estagiários e supervisores com filtros, busca instantânea e painéis de detalhes.

Frequência e atividades

Controle visual de presença, ausência, horários e atividades realizadas das 08:00 às 14:00.

Relatórios

Relatórios gerais, individuais e exportação PDF com jsPDF para documentação acadêmica.

Tecnologias

Construído para funcionar de forma totalmente estática.

  • HTML5, CSS3 e JavaScript ES6+
  • Bootstrap 5, Chart.js, Lucide Icons e jsPDF
  • LocalStorage para persistência local por navegador
  • Caminhos relativos compatíveis com GitHub Pages
Privacidade

O que está publicado e o que permanece privado.

Publicado

Interface, navegação, dados simulados, relatórios, configurações locais e estrutura estática para portfólio.

Privado

Projeto principal do estágio, regras internas, dados reais, decisões institucionais e qualquer informação sensível.

Persistência

Alterações feitas pelo visitante ficam apenas no próprio navegador por meio do LocalStorage.
